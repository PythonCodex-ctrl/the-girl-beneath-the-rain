# The Girl Beneath the Rain

A romantic storytelling website created as a birthday gift.

## Overview

"The Girl Beneath the Rain" is a single-page responsive website that tells the fictional love story of Joseph and Aria, inspired by real emotions and written as a heartfelt birthday surprise.

## Features

- 📖 Multi-chapter romantic story
- ❤️ Personal letter at the end
- 🎵 Background music
- 📱 Responsive design for desktop and mobile
- ✨ Smooth scrolling and animations
- 🔝 Back-to-top button

## Built With

- HTML5
- CSS3
- JavaScript
